# Sistema de CRUD Running Group
- Sitema criado com o objetivo de testar meus conhecimentos em CRUD, utilizei como tema a corrida de rua para a criação do sistema.

## Funcionalidades

- O sistema permite você cadastrar, consultar, atualizar e deletar **Grupos** ou **Acessorias** de corrida.


### Front-End:
- JavaScript
- React
- Vite

### Back-End:
- Node
- Express

### Banco de dados:
- MySQL

### Bibliotecas

- **Yup**: Utilizada para a criação esquemas destinados a validação de formulários.
- React-Hook-Form
- O **Yup** e o **React-Hook-form** trabalham juntos.

### Hooks React:

- UseState
- UseEffect
